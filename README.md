# Docker TODO-List

# Contexto
Projeto feito para reforçar aprendizados de comandos Docker, Dockerfile, Docker-compose.

## Técnologias usadas

Front-end:
> Desenvolvido usando: React, Redux, CSS3, HTML5, Redis, ES6

Back-end:
> Desenvolvido usando: NodeJS, ExpressJS, MongoDB, MYSQL, ES6


## Instalando Dependências

> Backend
```bash
cd api/ 
npm install
``` 
> Frontend
```bash
cd src/
npm install
``` 
## Executando aplicação

* Para rodar o back-end:

  ```
  cd api/ && npm start
  ```
* Para rodar o front-end:

  ```
    cd src/ && npm start
  ```
